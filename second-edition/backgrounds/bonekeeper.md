# שומר עצמות

> אתה רועה של הנפטרים. אתה מקשיב ללחישות האחרונות של המתים כשהם יורדים אל האדמה הקרה והבלתי מתפשרת. אתה יודע שכדי לחגוג באמת את מתנת החיים, עלינו לכבד גם את הפינאלה שלהם.

## שמות

רוק, אבון, מורו, אשוח, פאל, לת, בי, ברנבי, ערב, לדר

## ציוד התחלתי

- 3ק6 מטבעות זהב
- מנות (3 שימושים)
- פנס
- פחית שמן (6 שימושים)
- יתד (ק6)
- שרשראות (3 מטר)

## מה לקחת מהמתים? הגרל ק6:

|       |                                                                                                                                                                                                                                                                                                                                      |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | **קמע בצורת עורב**. אתה יכול לשאול שאלה מהמתים אך חייב להוסיף **תשישות** בכל פעם. הם לא תמיד מדברים אמת.                                                                                                                                                                                              |
| **2** | **פצע קטלני** מ_רוח נקם_ משוחררת. רופאת, אך ההשחתה הפכה אותך למנודה. אתה לא צריך לא אוויר ולא מזון אך עדיין כפוף לכאב ומוות. לכוד בין העולמות, המתים רואים אותך כאחד מהם.                                                                              |
| **3** | **דלי דם** (_מגושם_) מכת מוות מקומית. רוקן אותו כדי להעלות משרת בנוי ממה שקבור מתחת, עם 6 ה"פ, 1 שריון, 13 כח, 11 מי, 4 רצ, ואגרופי רסיסים (ק8+ק8). ניתן להעלות רק משרת אחד בכל פעם. אם הוא נהרס, אתה מאבד לצמיתות ק4 כח. **טעינה מחדש**: מלא בדם של לוחם גוסס. |
| **4** | **עגלת קבורה** (+6 משבצות, _איטי_) מהעבודה האחרונה שלך. היא הגיעה עם **חמור** זקן ועקשן (+4 משבצות, רק +2 משבצות אם מושך עגלה).                                                                                                                                                                                         |
| **5** | ספר הלחשים **גילוי קסם**, נגנב מספרייה עתיקה. משפחתך עבדה בשירות אלוהות עולם תחתון אלומה, אך איבדת את האמונה שלך. למרות שגורשת, אתה ממשיך לשרת, אפילו ככופר. גילוי קסם: אתה יכול לראות או לשמוע הילות קסומות בקרבת מקום. _נעשה חם למגע כאשר נעשה שימוש בקסם בקרבת מקום._     |
| **6** | **מסכת רופא מגפות**, אחרי שבעליה נפל קורבן למחלה שחיסלה את כל מי שהכרת פעם. הם היו צריכים להשאיר אותה עליהם.                                                                                                                                                                                             |

## איזה כלי היה יקר ערך בעבודתך? הגרל ק6:

|     |               |                                                                                                                            |
| --- | ------------- | -------------------------------------------------------------------------------------------------------------------------- |
| 1   | **אזיקים**  | למרות שישנים, הם עדיין יעילים אפילו נגד החזקים מאוד. אין לך את המפתח.                                     |
| 2   | **ספוג**    | כנראה עשוי משרידים של יצור ים נדיר. הוא נראה שאף פעם לא מתייבש.                                        |
| 3   | **גלגלת**    | מצוינת להזזת מצבות, סלעים, או אפילו גופות.                                                                       |
| 4   | **קטורת**   | מושלמת לטקסים או להרחקת זבובים. מקררת את הדם.                                                         |
| 5   | **מקוש**   | נזק ק6. לפעמים אתה פשוט צריך לפתוח את הדבר הארור!                                                             |
| 6   | **דוחה** | חומר חזק. התווית הדהויה שלו לא מבהירה נגד מה הוא _אמור_ להדוף, אבל. אולי הכל. 3 שימושים. |
