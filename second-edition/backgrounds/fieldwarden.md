# שומר השדות

> מגני היבול, הגנה מפני מזיקים, גנבים, וחיות. תפקיד של כבוד גדול, כל עוד הוא נמשך: שומרים רבים לא חיים את חייהם הטבעיים. הגרל פעם נוספת על טבלת **הקשרים**.

## שמות

זרע, דש, חורש, זורע, גבעול, מחרשה, קלח, פשתן, קוץ, שיפון

## ציוד התחלתי

- 3ק6 מטבעות זהב
- מנות (3 שימושים)
- לפיד (3 שימושים)
- בריגנדין (1 שריון, _מגושם_)
- קלע (ק6)
- גרזן יד (ק6)
- דוחה (בחר את הסוג, 3 שימושים)

## מה גבר עליך? הגרל ק6:

|       |                                                                         |
| ----- | --------------------------------------------------------------- |
| **1** | נחיל רעבתני של מזיקים שבלע יבולים וחיות כאחד. בלי מה להגן, עזבת. קח **תמצית זרע סערה** (3 שימושים). בליעתה מאפשרת לך לרוץ במהירות פי ארבע מהרגיל. אחרי כן אתה מוסיף שתי **תשישויות**.        |
| **2** | רוח יבול, שנכעסה מתרומה גרועה. האש כילתה כמעט הכל, ואחרי כן הצלחת לאסוף שקיק **זרעי אש** (ק8, _פיצוץ_, 4 שימושים).        |
| **3** | שד עם קרניים ושיניים שכמעט סיים אותך. קח **סכין עצם** מוכתמת בדם (ק6). ב**נזק קריטי**, ההתקפה הבאה שלה הופכת ל_מוגברת_ ממגע עם דם. |
| **4** | _הכמישה_, סוג של ריקבון גבעולים מה**שורשים**. קח **יבול נגוע** (6 שימושים) שמרקיב במהירות כל צמח שהוא נוגע בו.                        |
| **5** | זאבים, או שכך חשבת. אתה עכשיו **איש זאב** [8 ה"פ, 15 כח, 14 מי, טפרים (ק6+ק6), נשיכה (ק8)]. הרצ שלך נשאר אותו דבר. אתה יכול _להפוך_ לפי רצונך (פעם ביום) אך חייב לעשות הצלת רצ כדי לחזור. כל מי שנשאר בחיים מהתקפותיך חייב לעשות הצלת רצ כדי להימנע מהדבקה. |
| **6** | גנבי יבול. לא כולם שרדו, אך היית במיעוט מספרי. התחל עם +ק4 ה"פ ו**חרב מעוקמת** (ק8).  |

## איזה כלי הציל את חייך? הגרל ק6:

|       |                    |                                                                                                                                               |
| ----- | ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **שוט גפן דם** | נזק ק8. ב**נזק קריטי**, הוא שואב את דם המטרה, מעניק להתקפה הבאה של הנשק את תכונת ה_פיצוץ_             |
| **2** | **שומר רעש** | מכשיר עם ידית סיבוב שפולט רעש חזק, מפחיד את רוב היצורים.                                                               |
| **3** | **מקל שמש**      | מספק חום ואור מספקים עד שעה. **טעינה מחדש**: השאר באור שמש כבד ליום שלם. שימוש 1. |
| **4** | **רצועת שורש**    | כשנזרקת, קושרת יצור בגודל זאב לאדמה לזמן קצר.                                                                        |
| **5** | **חליל ירוק**   | חליל קטן שמרגיע צמחים, הופך את המעבר דרך אזורים עמוסי צמחייה לקצת יותר קל.                                                   |
| **6** | **כתר פריחה נצחי** | עטרה מעוטרת בפרחים שאף פעם לא נובלים. ב**נזק קריטי**, הפרחים מתפוררים לאבק, אך אתה פועל כאילו ההצלה שלך הצליחה (אובדן כח עדיין מתרחש). |
